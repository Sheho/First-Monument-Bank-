# GrandCap Banking Platform

GrandCap is a modern, full-featured online banking platform built with React, Vite, and Tailwind CSS.

## Features

- **Account Management:** Securely log in to access your dashboard.
- **Multiple Account Types:** View and manage Checking and Savings accounts.
- **Loans:** Explore and apply for Auto Loans and Personal Loans.
- **Money Transfers:** Easily transfer funds between accounts or to other users.
- **Virtual Cards:** Apply for and manage secure virtual debit/credit cards instantly.

## Development

This application is built with React and Vite.

### Running Locally

```bash
npm install
npm run dev
```

### Build for Production

```bash
npm run build
```
